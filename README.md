## About
Daily Task Flow is built to help users manage tasks efficiently and maintain focus throughout the day. Its intuitive design lets you add, track, and complete tasks without distractions.

---

## Features
- Add, edit, and remove daily tasks  
- Mark tasks as completed  
- Clean and minimal user interface  
- Fully responsive design for mobile and desktop  
- Fast performance with simple navigation

## Demo

## Usage
1. Open the app in your browser  
2. Add your daily tasks in the input field  
3. Mark tasks as completed when done  
4. Remove tasks you don't need  
5. Stay organized and focused
